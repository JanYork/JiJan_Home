# 极简团队主页
 HTML5和CSS3以及少量JavaScript代码构成的轻量级团队主页

# 开发说明
对于这个前端页面，我们没有后端，他并不是特别完美，所有改动须在源代码中修改
而他的目的也并非使用此页面，而是让前端入门的新手从中获取到大量的前端技术与支持
